# Titanic-Survival-Prediction-Logistic-Regression

This project predicts whether a passenger survived the Titanic disaster or not.

## What I used
- Logistic Regression (Binary Classification)
- Python, Pandas, Scikit-learn, Matplotlib

## What the model does
- Takes passenger details like age, gender, ticket class
- Predicts: Survived (1) or Not Survived (0)
